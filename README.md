# SSM桂电书籍管理系统

#### 介绍
SSM桂电书籍管理系统，使用SSM框架练手的轻量级图书管理系统，项目非常简单，适用于刚入门的新手学习。

#### 软件架构
- **环境依赖**

  Tomcat 8.5.60

  Mysql 8.0.23

  JDK13

- **使用技术**

  SpringMVC

  Mybatis

  JSP
